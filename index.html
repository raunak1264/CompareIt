Help & Instructions

Getting Started

This tool allows you to compare two pieces of code or text side by side and highlight the differences between them.

Basic Usage

  1. Paste or type code/text into both panels
  2. Select the appropriate language from the dropdown
  3. Click "Compare Differences" to see the changes highlighted

Features

  • Language Selection: Choose the language for syntax highlighting
  • View Modes: "Side by Side" or "Unified Diff" view
  • Line Numbers: Toggle line numbers on/off
  • File Loading: Load files directly from your computer
  • Download: Save the comparison results

Keyboard Shortcuts

  • Ctrl+Enter: Compare differences
  • Ctrl+L: Toggle line numbers
  • Ctrl+R: Reset both editors
